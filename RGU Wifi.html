Wifi Connection

Students who wish to connect to the free
wifi on campus can do so by using any of
these wifi connections.

  1. Eduroam
  2. The Cloud

Eduroam Connection Guide

Setting up a connection to Eduroam is easy

1. Go to settings > Connections wifi
2. Select eduroam from the list of
wifi > connect to network
3. Enter your RGU credentials as below
  • EAP method - PEAP
  • Phase-2 Authentication - MSCHAPV2
  • a.surname@rgu.ac.uk(i.e your school email)
  • password (You school password)
4. You will then be connected to eduroam

The Cloud

For external visitors without Eduroam
The cloud wifi service is availble across
Garthdee and the city center.

Connection Guide


1. Select the cloud Network from available
network list
2. Open your internet browser - The Cloud
landing page will appear. if it is your first
time using the cloud wifi network, follow the
simple one-time registration process or login


The Cloud offers free internet access for 3 hours
total per day per device on the RGU campus (except
where special access may be arranged for event.)
Student should use the secure and constant free
Eduroam wireless, service instead.

Other accessible Wifi in Aberdeen can be firstbus wifi, but its you must be inside a firstbus
